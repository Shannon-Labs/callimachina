Bayesian Confidence Enhancement

Traditional reconstruction methods achieve ~50-60% confidence through qualitative assessment. CALLIMACHINA integrates multiple evidence streams using Bayesian updating:

P(Authentic | Genre) = 0.50

# Evidence integration

üíª API & Usage

Simple Python API for integrating CALLIMACHINA into your research workflow.

Quick Start

from pinakes.integration_engine import IntegrationEngine

# Initialize engine
engine = IntegrationEngine(verbose=True)

# Reconstruct a lost work
results = engine.run_full_pipeline(
    target_works=["Eratosthenes Geographika"],
    confidence_threshold=0.95
)

# Access results
reconstruction = results["Eratosthenes Geographika"]
print(f"Confidence: {reconstruction['confidence']:.1%}")
print(f"Fragments: {len(reconstruction['fragments'])}")

# View evidence breakdown
for factor, score in reconstruction['evidence_factors'].items():
    print(f"{factor}: {score:.1%}")

Command Line Interface

# Run full pipeline
python3 pinakes/integration_engine.py

# Reconstruct specific work
python3 callimachina/src/cli.py --reconstruct "Eratosthenes Geographika"

# Generate report
python3 callimachina/src/cli.py --report --format markdown

# Custom configuration
python3 callimachina/src/cli.py --config config/custom.yml --verbose
