## üîç METHODOLOGY: HOW WE DID THIS

### Bayesian Confidence Framework

Each reconstruction is a probability distribution, not a single text:

```
Posterior ‚àù Prior √ó Likelihood

Prior: Based on genre, century, author fame
  P(work exists | metadata) ~ Beta(Œ±, Œ≤)

Likelihood: Based on fragment evidence
  P(fragments | work exists) ~ Bernoulli(p)

Posterior: Updated belief
  P(work exists | fragments) ‚àà [0, 1]
```

**Key Innovation**: We don't guess the text - we quantify our uncertainty about it.

### Citation Network Analysis

**Central Question**: If a work is cited by N authors but has no surviving copies, what's the probability it existed?

**Answer**: 
```
recoverability = 1 - (1 - p_survival)^N

where p_survival = f(century, genre, author_fame)
```

**Discovery**: Works cited by >3 authors have >85% probability of having existed, even with no surviving manuscripts.

### Stylometric Author Discovery

**Method**: 34-dimensional feature space (vocabulary, syntax, morphology, character patterns)

**Algorithm**: DBSCAN clustering with Œµ=2.0, min_samples=2

**Result**: 6 works cluster as outliers ‚Üí potential new authors

**Validation**: Unknown.TragicPoet1 has unique metrical signature not matching any known tragedian.
